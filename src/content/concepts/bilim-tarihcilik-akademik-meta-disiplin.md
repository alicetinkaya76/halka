## Atomik özet

**Bilim-tarihçilik akademik-meta-disiplini**, Halka 4. dönem-müfredat-tasarımının (Karşılaştırmalı Sentezler / Modern Bilim ve Bilim Tarihçiliği akademik-yan-konum) akademik-meta-yöntem-felsefe-yapılaşmasının yapısal yan-zeminidir: Kuhn (Halka #18) → Sarton (Halka #19) → Gavroglu (Halka #20) → Ronan (Halka #21) akademik-zincirin yapısal yan-yana-yapılaşmasının disiplinli akademik-mimarisidir. Halka 1.-3. dönem-akademik-zincirinin (genel-bilim-tarihi + Doğu-bilim-İslâm-zincir + Batı-Bilim-Devrimi) akademik-meta-yapılaşmasının disiplinli yansımasıdır — bilim-tarihçilik-akademik-disiplin-akademik-meta-konum-zincirin somutlaşması.

## Halka 4. dönem-akademik-meta-zincirin yapısal-anatomisi

Halka 4. dönem-akademik-meta-yöntem-felsefe-zincirin disiplinli yan-yan-yapılaştırılması:

- **#18 Kuhn (1962/1969 *Bilimsel Devrimlerin Yapısı*)** — **akademik-meta-felsefe-yapılaşma-yan'ı**. Halka 4. dönem-açılış-vakası. Paradigma-değişim-akademik-yöntem-zincirin disiplinli formülasyonu (paradigma → olağan-bilim → aykırılık-birikimi → kriz → bilimsel-devrim → yeni-paradigma döngüsel-akademik-yapısı). Bilim-tarihçilik-akademik-disiplinin akademik-meta-felsefe-yan-yapılaşması.

- **#19 Sarton (1916-1956 *Bilim Tarihinde Yöntem*)** — **akademik-tarihçi-disiplin-formülasyon-yan'ı**. Halka 4. dönem-ikinci-vakası. Bilim-tarihçilik-akademik-disiplinin Amerikan-kurucu-formülasyonu (1913 *Isis* + 1924 History of Science Society + *Introduction* 5-cilt-akademik-derinlik-monografi-vakası 1927-1948). Hümanist-bilim-tarihi-akademik-evrensel-pozitif-zemini.

- **#20 Gavroglu (*Bilimlerin Geçmişinden Tarih Üretmek*)** — **akademik-eleştirel-eklemlenme-yan'ı**. Halka 4. dönem-eleştirel-rafinasyon-vakası. Sarton-akademik-pozitif-konumlamasının akademik-disiplinli iç-eleştirisi; Hessen-Marksist-dışsalcı-akademik-konumlamasının disiplinli akademik-yenidoğuş-vakası — eleştirel-eklemlenme-tezi-akademik-disiplinli yapılaşması.

- **#21 Ronan (*Bilim Tarihi*)** — **akademik-panoramik-bilim-kültür-anlatımı-yan'ı**. Halka 4. dönem-kapanış-vakası. Çoklu-medeniyet-eksenli yaklaşım, panoramik-popüler-akademik-üslupta yapılaşma — Halka 4. dönem-akademik-zincirin akademik-pedagojik-kapanışın yapısal yan'ı.

## Halka 1.-3. dönem-akademik-zincirinin akademik-meta-yapılaşması

Halka 4. dönem-akademik-meta-disiplin-zincirin yapısal-yan-zemini Halka 1.-3. dönem-akademik-zincirinin disiplinli akademik-meta-yapılaşmasıdır:

- **Halka 1. dönem (#1-4: Fara, Goody-BD, Goody-R, Hobson)** — genel-bilim-tarihi + Doğu-bilim-zincir-açılış-yan'ı; Halka 4. dönem-akademik-meta-disiplinin yapısal-akademik-zemini olarak yapılaşır (anti-Eurocentric-akademik-konumlamasının yapısal yan-zemini)
- **Halka 2. dönem (#5-11: Goody-BD, Starr, Pedersen, Berkey, Sezgin, Lyons, Gutas)** — Doğu-bilim-İslâm-zincirin akademik-derinlik-yan'ı; Halka 4. dönem-akademik-meta-disiplinin Sarton-İslâm-bilim-zincir-akademik-takdir-konumlamasının disiplinli akademik-mirasçı-yan'ı
- **Halka 3. dönem (#12-17: Gleick, Wulf, Isaacson-Franklin, Westfall, Isaacson-Einstein, Wootton)** — Batı-Bilim-Devrimi figür-merkezli-popüler-akademik-biyografi-katmanı + akademik-derinlik-monografi-katmanı + akademik-tarihçi-disiplinli-derinlik-yenidoğuş-katmanı çift-yapılaşması; Halka 4. dönem-akademik-meta-disiplin akademik-tarihsel-yapılaşmasının yapısal yan-zemini

## Halka için akademik-yapısal-yan'ı

Bilim-tarihçilik akademik-meta-disiplin atomiği Halka 4. dönem-müfredat-tasarımının akademik-meta-konumlamasının somut-akademik-vakasıdır. Halka grup-okumasında bu meta-konum-zincir-yapılaşmasının disiplinli yan-yan-okunması:

- **Kuhn-Sarton yan-yan-okunması:** akademik-meta-felsefe × akademik-tarihçi-disiplin-formülasyon yan-yan-konum-rafinasyonu (Halka #18-#19 yan-yana-okuma-zemini)
- **Sarton-Gavroglu yan-yan-okunması:** akademik-evrensel-pozitif-konum × akademik-eleştirel-eklemlenme yan-yan-konum-rafinasyonu (Halka #19-#20 yan-yana-okuma-zemini)
- **Gavroglu-Ronan yan-yan-okunması:** akademik-eleştirel-konum × akademik-panoramik-pedagojik-konum yan-yan-konum-rafinasyonu (Halka #20-#21 yan-yana-okuma-zemini)

Halka grup-okumasının akademik-meta-yapılaşma-akademik-okuma-zinciri Halka 4. dönem-akademik-zincirin yapısal yan-yan-konumlamasında akademik-disiplinli-yan-yana-yapılaşmasında akademik-zenginleşir.

## Türkçe-yan-zemin: Türkiye-bilim-tarihçilik-akademik-zincir

Halka için **Türkçe-akademik-meta-zincir-vakası**: Türkiye-bilim-tarihçilik-akademik-zincirinin Sarton-Sayılı-DTCF-akademik-meta-mirası (1942-1952-1990'lar+) Halka 4. dönem-akademik-meta-disiplin-zincirin Türkçe-yansıma-akademik-yan'ıdır. Türkçe-akademik-meta-disiplin-zincirin somutlaşması: 1995 Gündoğan-*Antik Bilim ve Modern Uygarlık* + 1997 Doruk-*Bilim Tarihinde Yöntem* (Halka #19) + 2007 Demir-*Türkiye'de Bilim Tarihi Araştırmalarının Tarihi* + 2020 Muhayyel-*Bilim Tarihi Araştırmalarında Yöntem* yenilenen 2. baskı.
